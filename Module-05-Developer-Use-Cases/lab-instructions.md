# Lab 5: Real-World Developer Use Cases

**Duration:** 30 minutes

## Objectives
- Apply Copilot to real development scenarios
- Practice feature development workflow
- Refactor legacy code safely
- Generate comprehensive documentation

## Exercise 1: New Feature Development (10 min)
Implement a complete shopping cart feature:
1. Plan with Copilot Chat
2. Generate service class
3. Create tests
4. Document API

## Exercise 2: Legacy Code Refactoring (10 min)
Refactor provided legacy code:
1. Understand with `/explain`
2. Generate tests first
3. Refactor incrementally
4. Verify with tests

## Exercise 3: Debugging Session (5 min)
Debug a complex issue:
1. Analyze error with Copilot
2. Add debug logging
3. Generate test cases
4. Fix the bug

## Exercise 4: Documentation Generation (5 min)
Create complete documentation:
1. Function documentation
2. API documentation
3. Usage examples
4. README generation

**Deliverables:**
- Completed shopping cart feature
- Refactored legacy code
- Fixed bug with tests
- Comprehensive documentation

## Key Takeaways
- Copilot accelerates development
- Always test refactored code
- Use Chat for complex debugging
- Documentation can be automated
