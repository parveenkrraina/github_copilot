# Lab 7: Multi-Language Development

**Duration:** 30 minutes

## Objectives
- Use Copilot effectively in C#
- Use Copilot effectively in JavaScript
- Apply language-agnostic patterns
- Adapt prompts across languages

## Exercise 1: C# Development (10 min)
Create a complete C# API:
1. ASP.NET Core controller
2. Service with dependency injection
3. Entity Framework queries
4. Unit tests with xUnit

## Exercise 2: JavaScript Development (10 min)
Create equivalent JavaScript API:
1. Express.js routes
2. Service layer
3. Database queries
4. Jest tests

## Exercise 3: Cross-Language Patterns (5 min)
Implement same pattern in both languages:
1. Factory pattern
2. Repository pattern
3. Observer pattern

## Exercise 4: Prompt Adaptation (5 min)
Take one prompt and use it in:
1. C#
2. JavaScript
3. Python
4. Compare results

**Deliverables:**
- C# API implementation
- JavaScript API implementation
- Cross-language pattern examples
- Adapted prompts documentation

## Key Takeaways
- Same prompt works across languages
- Copilot adapts to language conventions
- Focus on concepts, not syntax
- Universal patterns translate well
