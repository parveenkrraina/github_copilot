# Lab 6: Test Generation Workshop

**Duration:** 30 minutes

## Objectives
- Master test generation with Copilot
- Create comprehensive test suites
- Identify edge cases systematically
- Use Playwright for E2E tests

## Exercise 1: Unit Test Generation (10 min)
Generate tests for provided functions:
1. Use `/tests` command
2. Add edge cases
3. Include negative scenarios
4. Verify coverage

## Exercise 2: Test Data Factories (5 min)
Create test data factories:
1. User factory
2. Order factory
3. Product factory

## Exercise 3: Integration Tests (5 min)
Generate integration tests:
1. API endpoint tests
2. Database interaction tests
3. Authentication tests

## Exercise 4: Playwright E2E Tests (10 min)
Create E2E tests with Playwright:
1. Login flow
2. Shopping cart flow
3. Checkout process
4. Page object model

**Deliverables:**
- Complete test suite with 80%+ coverage
- Test data factories
- E2E test scenarios
- Page objects

## Key Takeaways
- `/tests` command accelerates testing
- Always include edge cases
- Test data factories improve maintainability
- E2E tests validate user flows
